# Auvora - Premium AI Prompt Library

A beautifully redesigned, professional web application for discovering and using AI prompts across 20+ categories and 125+ niches.

## ✨ Features

- **Premium Design**: Dark theme with sophisticated gradients, shadows, and animations
- **Live on GitHub Pages**: https://gamingone7233-eng.github.io/Ai-prompt/
- **Custom Domain**: auvora.store (via CNAME)
- **1,000,000+ Prompts**: Generated from 5,000+ expert templates
- **20 Categories**: Hooks, Scripts, Ideas, Captions, Copywriting, Email, SEO, Branding, Strategy, Video
- **125 Niches**: From Fitness to Technology to Cooking and beyond
- **12 Platforms**: YouTube, TikTok, Instagram, and more
- **Interactive Features**: Search, filter, copy to clipboard, shuffle
- **Responsive Design**: Works perfectly on all devices
- **Bonus Content**: Ready hooks, viral breakdowns, 30-day plan, monetization guide

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser or serve with a local server
3. Start exploring prompts!

For local development:
```bash
python3 -m http.server 8000
# Then visit http://localhost:8000
```
